# Многоязычная реализация (i18n) - Сводка

## Обзор
IPv4 Subnet Calculator теперь полностью поддерживает три языка:
- **ENG** - English (Английский)
- **РУС** - Русский (Russian)  
- **ARM** - Հայերեն (Armenian)

## Архитектура i18n

### 1. Context API (`src/contexts/I18nContext.jsx`)
- Централизованное управление состоянием языка
- Функция `t(key)` для перевода с поддержкой вложенных ключей ("section.key")
- Функция `changeLanguage(lang)` для переключения языков
- Сохранение выбранного языка в `localStorage`
- Автоматическое восстановление предыдущего выбора при загрузке

### 2. Файлы переводов (`src/locales/`)
- **en.json** - Английский переводы (350+ ключей)
- **ru.json** - Русский переводы  
- **hy.json** - Армянский переводы

**Структура ключей:**
```json
{
  "title": "...",
  "calculator.mode": "...",
  "network.information": "...",
  "result.error": "...",
  "modal.explanation": "...",
  "formula.networkAddress": "...",
  "binary.show": "..."
}
```

## Интегрированные компоненты

### SubnetCalculator.jsx
- Заголовок с селектором языка (кнопки ENG/РУС/ARM)
- Все текстовые метки переведены
- Примеры ввода на каждом языке
- Кнопка режима тёмной темы рядом с переключателем языков

**CSS для языков:**
```css
.lang-btn { /* Базовый стиль */ }
.lang-btn.active { /* Активная кнопка */ }
```

### ResultDisplay.jsx
- Все названия полей результатов переведены
- Поддержка обоих режимов: Standard Subnetting и VLSM
- Метки для всех IP адресов, масок, префиксов
- Описание алгоритма VLSM на выбранном языке

### ExplanationModal.jsx
- Интерактивные модали с объяснениями
- 7 типов объяснений: firstIp, lastIp, subnetMask, usableHosts, networkMask, networkAddress, broadcastAddress
- Полностью переведены:
  - Названия модалей
  - Объяснения
  - Шаги формул
  - Примеры расчетов

### App.js
- Обёрнут `I18nProvider` для глобальной доступности переводов

## Использование переводов в компонентах

```jsx
import { useI18n } from '../contexts/I18nContext';

function MyComponent() {
  const { t, language, changeLanguage } = useI18n();
  
  return (
    <div>
      {/* Перевод простого ключа */}
      <h1>{t('title')}</h1>
      
      {/* Перевод вложенного ключа */}
      <label>{t('network.baseIpAddress')}</label>
      
      {/* Переключение языка */}
      <button onClick={() => changeLanguage('en')}>English</button>
      <button onClick={() => changeLanguage('ru')}>Русский</button>
      <button onClick={() => changeLanguage('hy')}>Հայերեն</button>
      
      {/* Получение текущего языка */}
      <p>Current: {language}</p>
    </div>
  );
}
```

✅ **Полное покрытие**
- 350+ строк переведены на три языка
- Все UI элементы включены

✅ **Сохранение языка**
- Выбор языка сохраняется в localStorage

✅ **Интерактивные нотации**
- Модали с объяснениями для каждого вычисления
- Многоязычные формулы и примеры

✅ **Мобильная оптимизация**
- Селектор языков покрывает все устройства
- Кнопки языков удобны для мобильных пользователей
